To use Foundation, you want to assign classes to individual HTML elements. For example, a ul can be made to work like a menu like this: < ul class="menu" >

Top Bar

  • The following is a < div class="top-bar" >, containing a < div class="top-bar-left" > and < div class="top-bar-right" >.
  • Quite predictably, the elements inside the left top bar start from the left, and vice versa for the right.
  • Both the left and right top bars contain a < ul class="menu" >. This is necessary because a normal ul goes from top to bottom. This makes the li elements go horizontally.
  • Left: the first li is a < li class="menu-text" >, which distinguishes it from the rest.
  • Right: you can see a < button class="button" >, which is a style of button defined in Foundation.

  Cards

  • The following is a < grid-container > of < div class="card" >.
  • The "grid-x" class lays the cards out horizontally.
  • Inside a card can be a < div class="card-divider" >, which serves well as a header.
  • Below the card divider is just an img element linking to the image.
  • Below the image is a < div class="card-section" >, which is visibly different from the card divider. You can put descriptive text in here.

  Tables

  • Tables work exactly the same as in HTML. The < thead > tag bolds the text and serves as the "head" of the table.
  • The table is of class="hover", which means that the row you are hovering over highlights.
  • By default, Foundation tables are striped for contrast. This can be modified by changing the class of the table.
